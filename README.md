# NETLIFT

## Installation

### Client

- Clone repo and `cd` into folder
- `cd client`
- `npm install` to install dependencies
- `npm install --save-dev` to install dev dependencies

### Server

- git submodule update --init
- `cd server `
- `pipenv shell`
- `pipenv install --dev`
- `pipenv run dev`

## Usage

### Client

- `npm run dev` to run in development mode
